# Dashboard Agroanálise - Power BI

Este projeto apresenta um dashboard interativo criado no Power BI com foco na análise da produção e exportação agropecuária brasileira nos anos de 2023 e 2024.

## Objetivos
- Analisar os volumes de produção e exportação.
- Visualizar percentuais exportados e de consumo interno.
- Identificar estados e regiões com maior destaque por produto.

## Tecnologias utilizadas
- Power BI
- DAX
- SQL (modelagem via views)
- Excel (como fonte de dados)

## Funcionalidades
- Filtros interativos por Produto e Ano
- Cartões com KPIs
- Gráficos de ranking por estado e região
- Análise de evolução anual da produção e exportação
